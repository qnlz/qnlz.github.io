应用table-layout:fixed之后的效果:
1、如果不指定任何宽度，则各列的宽度是平均分配的；
2、后序的表格行并不会影响列宽
3、给单元格指定很大的宽度也会直接生效，并不会自动缩小；
4、overflow 和 text-overflow属性都是可以正常生效的；
5、如果overflow的值是visible，则单元格的内容有可能会溢出；
